# Student_Dashboard
A python-based dashboard for analyzing student marks with statistics and visualizations.
Built with Python and Streamlit, this project demonstrates data handling, statistical analysis, and basic web app development.

#features

Add student names and marks
Calculate average, minimum, and maximum scores
Display class pass/fail status
Identify top-performing student
Visualize performance using graphs

#How to run

	1.	Install dependencies:
      pip install streamlit matplotlib
	2.	Run the app:
      streamlit run app
